        线
        imageline(resource $img,int $x1,int $y1,
                           int $x2,int $y2,int $color)
         功能：画线 
         参数：
         img 创建画布时申请的资源
         x1,y1 线的起点的坐标
         x2,y2 线的终点的坐标
         color 线的颜色

         矩形
         imagerectangle(resource $img,
                                     int $x1,int $y1,
                                     int $x2,int $y2,
                                     int $color )
         功能：矩形
         参数：
         img 申请的画布资源
         x1,y1 矩形在画布上的起点坐标
         x2,y2 矩形在画布上的终点坐标 
         color  矩形边框的颜色

           imagestring(resource $img,int $font,
                                 int $x,int $y,string $s,
                                 int $color)
           功能：输出文字
           参数：
           img 申请的画布资源
           font 字体大小（1~5）1 最小，5 最大
           x,y   字的起点位置
           s       输出的文字
           color 字体的颜色

           imagettftext(resource $img,
                                   int $font,float $angle,
                                   int $x,int $y,
                                   int $color,string $ttffile,
                                   string $s)
          功能：输出文字
          参数：
           img  申请的画布资源
           font 文字的大小。 C:\WINDOWS\Fonts
           angle 角度，水平是0度，逆时针旋转。
           x,y     文字的起点坐标
           color  文字的颜色
           ttffile 字体文件的名字
           s         输出的文字
