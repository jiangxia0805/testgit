GD库
一、概述
       生成动态图片。
       
        GD库的主要用途
         - 验证码
         - 动态图表
         - 水印
         - 图片的压缩和裁剪

二、安装
       - 确认安装否
          phpinfo() 输出php的配置信息
       -  改php.ini文件
           A、extention_dir = php安装所在目录下的ext目录。
           B、extension=php_gd2.dll去掉前面的“；”
　　－重启网络服务器(apache)
        - 再次确认是否安装gd库（phpinfo）

三、应用
       1、纸
             imagecreatetruecolor(int $width,int $height)
             功能：创建画布
             参数：
             width 画布的宽
             height 画布的高
             返回：资源

       2、画
             imagecolorallocate(resource $img,
                                                int $red,
                                                int $green,
                                                int $blue)
             功能：创建颜色
             参数：
             img  申请的画布资源
             red   红色 （0~255）
             green 绿色 （0~255）
             blue   蓝色  （0~255）
             返回 颜色值

             imagefill(resource $img,
                              int $x,int $y,
                              int $color)
             功能：给画布填充背景色
             参数：
               img  申请的画布资源
               x，y  坐标
               color 填充的背景色
 

       3、展示/收藏
            展示：
            - 通知浏览器将要输出图片
               header("Content-Type:image/gif")
            - 输出图片
               imagegif(resource $img)
               功能：输出图片
               参数：
               img 申请的图片资源


            - 通知浏览器将要输出图片
               header("Content-Type:image/png")
            - 输出图片
               imagepng(resource $img)
               功能：输出图片
               参数：
               img 申请的图片资源

            - 通知浏览器将要输出图片 
               header("Content-Type:image/jpeg")
            - 输出图片
               imagejpeg(resource $img)
               功能：输出图片
               参数：
               img 申请的图片资源

             收藏或者保存
             imagegif(resource $img,string $filename) 
             imagejpeg(resource $img,string $filename) 
             imagepng(resource $img,string $filename)
             功能：
             保存图片
             参数：
             img 申请的图片资源
             filename 指定要保存的图片文件名称。
                            注意：  保持图片扩展名跟函数名称保持一致。





       4、清理
             imagedestroy(resource $img)
             功能：释放画布资源
             参数：
             img 申请的画布资源
             返回 ：成功 true 
                         失败 false
